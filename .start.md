## 🎨 UI 框架升级

### 已集成 Naive UI 框架

**安装的依赖:**
- `naive-ui^2.43.2` - 主 UI 框架
- `@iconify/vue^5.0.0` - 图标库支持
- `@vicons/ionicons5^0.13.0` - IonIcons 5 图标集
- `unplugin-vue-components^31.0.0` - 按需引入组件
- `unplugin-auto-import^21.0.0` - 自动导入 API

**配置方式:**
- 按需引入：只使用实际需要的组件
- 自动导入：常用 API 不需要显式导入
- 类型支持：完整的 TypeScript 支持

### Naive UI 组件使用示例

#### 表单组件
```vue
<n-input v-model:value="form.username" placeholder="请输入用户名" />
<n-input-number v-model:value="form.rent" :min="0" />
<n-select v-model:value="form.type" :options="options" />
<n-date-picker v-model:value="form.date" type="daterange" />
<n-button type="primary" @click="handleSubmit">提交</n-button>
```

#### 数据展示
```vue
<n-data-table :columns="columns" :data="list" :pagination="pagination" />
<n-card title="房屋信息" :bordered="false">
  <!-- 卡片内容 -->
</n-card>
<n-statistic label="总收入" :value="totalIncome">
  <template #prefix>￥</template>
</n-statistic>
```

#### 布局组件
```vue
<n-layout has-sider>
  <n-layout-sider>
    <n-menu :options="menuOptions" />
  </n-layout-sider>
  <n-layout-content>
    <!-- 主内容 -->
  </n-layout-content>
</n-layout>
```

#### 反馈组件
```vue
<n-dialog-provider>
  <n-modal v-model:show="showModal" title="标题">
    <!-- 模态框内容 -->
  </n-modal>
</n-dialog-provider>

  <n-button @click="showMessage">显示消息</n-button>

```

### 按需引入配置

在 `vite.config.ts` 中已配置：

```typescript
Components({
  resolvers: [NaiveUiResolver()],
  dts: true,
}),
AutoImport({
  imports: ['vue', 'vue-router', 'pinia'],
  dts: true,
}),
```

自动导入的 Naive UI APIs:

- useMessage
- useNotification
- useLoadingBar
