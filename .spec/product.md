# 产品规格文档

## 产品名称

个人租房管理系统 (Rental Management System)

## 产品定位

一个轻量级、易使用的个人房东租房管理系统，专为单人使用设计，不做 SaaS 化。

## 目标用户

- 个人房东（拥有少量房屋出租）
- 房产托管人员
- 小型中介公司（单人管理）

## 产品价值

- 简洁：界面简洁，操作直观
- 易维护：代码结构清晰，便于后续开发
- 易部署：Docker 一键部署
- 稳定：适合个人长期使用

## 核心功能

### 1. 登录系统
- 单管理员账号配置
- JWT Token 认证
- 租户可选只读账号

### 2. 房屋管理
- 房屋信息 CRUD
- 状态切换（空置/已租）
- 房屋详情展示

### 3. 租户管理
- 租户信息 CRUD
- 自动绑定房屋
- 退租自动释放房屋

### 4. 缴费记录
- 多种缴费类型（房租/水费/电费/其他）
- 缴费时间筛选
- 月度/年度统计

### 5. 数据统计
- 本月/年度收入
- 收入趋势图
- 费用占比图
- 出租率统计

## 产品限制

- 单人使用，不支持多租户
- 不做 SaaS 化
- 无复杂的权限管理
- 适合管理 1-100 套房屋

## 非功能需求

### 性能
- 页面加载时间 < 2s
- API 响应时间 < 500ms

### 兼容性
- 支持现代浏览器（Chrome/Firefox/Safari/Edge 最新版本）
- 移动端 iOS 14+ / Android 10+
- 响应式布局，适配各种屏幕尺寸

### 安全性
- JWT 认证
- 密码 bcrypt 加密
- SQL 注入防护
- XSS 防护

### 可用性
- 7x24 小时稳定运行
- Docker 容器化部署
- 支持自动重启

## 技术架构

### 前端
- Vue 3 + TypeScript
- Vite 构建工具
- Vue Router 路由
- Pinia 状态管理
- Axios HTTP 客户端
- TailwindCSS 样式框架
- ECharts 图表库
- 移动端优先响应式设计

### 后端
- NestJS 框架
- Prisma ORM
- PostgreSQL 数据库
- JWT 认证
- Swagger API 文档
- 全局异常处理
- 日志拦截器

## 部署要求

- Docker 20.10+
- Docker Compose 2.0+
- 2GB 内存
- 20GB 磁盘空间
